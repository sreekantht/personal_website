Robot Mr. Black

This autonomous machine mechanically turned on/off 9 candlelight LED and report their quality. It has 3 subsystems: Electromechanical, Electrical and Micro-controller. It was designed, machined and fabricated over the course of 4 months by 3 "engineers".

Wireless Robot Controller

Using 2 Xbee + Ardunio Uno Combo to implement a wireless E-stop for an autonomous rover. One Xbee is controlled by a push button. The other Xbee is connect to Op-Amp and Relay circuit. When user press down the button. It cuts off the power from battery.

Smart Eyeglass

A low-fatality prototype of an eyeglass that is capable of detecting sound from surrounding traffic. It is used to alert the users (which is targeted to be deaf cyclists) which way are the cars coming from. The prototype is built on Arduino Leonardo with a microphone.

ThoughtSpot App

A Hack from the ThoughtSpot Hackathon. It is an android app that used a backend built on Django and served on Google App Engine. It used data provided by ThoughtSpot to give a better UI/UX experience to users with additional function.

AutoComis

A Wearble Hack from N36 Wearable Hackathon. It integrates the Myo with a MeU wearable LED array.It enhances the communication between drivers by displaying emoticon on MeU based on gesture. Won 3rd place.

Data Acquision System

Coded and optimized the data acquision system of a lab apparatus at U of T. Included a new user interface, new control of LED based on Camera action and take data from several new sensors. The data system acquision is coded in Labview.

Notabilia Website

A website built on Bootstrap, PHP, MySQL, HTML5, CSS3 and jQuery. It is a website for a tutoring school, Notabilia Ltd. SEO is also implemented. The site is still under construction.

Personal Website

onTrack

Hospital Patient Organizer
